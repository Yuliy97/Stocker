# d3-stock

Stock prices visualization using d3.js

See, all in one chart, with multiple time spans (1wk, 1mth, 6mths, 1yr, 5yr):
* actual stock prices
* a mobile average
* trading volumes

**Demo:** http://arnauddri.github.io/d3-stock/
